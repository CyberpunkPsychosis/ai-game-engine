# 音效挂载点

把对应文件放进本目录（`.wav`），游戏会自动播放；**没有文件就静默**，不报错。
所以你随时丢音效进来即可，无需改代码。

| 文件名 | 触发时机 |
|---|---|
| `hit.wav` | 命中敌人 |
| `enemy_die.wav` | 敌人死亡 |
| `slash.wav` | 挥砍/连击/挑空 |
| `slam.wav` | 俯冲砸地 |
| `dash.wav` | 冲刺 |
| `jump.wav` | 跳跃 |
| `cast.wav` | 大招孢爆 |

> 来源建议：freesound.org、付费音效包、或 AI 音效。统一音量/风格后丢进来。
> 想加新触发点，告诉我事件名，我在代码里挂一行 `FX.sfx("名字")`。
