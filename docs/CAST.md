# 角色与资产花名册 · 弹反 Roguelite

> 全部"要做/要找的角色与资产"清单。⭐=垂直切片必做;其余为完整版(Steam 体量)。
> 动画帧数规格见 `CHARACTER_POSES.md`;敌人攻击行为见 `ENEMY_DESIGN.md`。主题皮待定,这里是"角色与功能"。

## 一、主角(1 名)⭐
被困轮回的剑客(《剑的轮回》)。**一套骨架,武器可换皮**。
- **必需动画**:`idle`、`走/跑`、`跳`、`下落`、`闪避/翻滚(无敌帧)`、`轻攻击(2–3 连段)`、`重攻击`、
  **`格挡/举刀架势`**、**`完美弹反(火花瞬间)`**、`处决(终结技)`、`受击`、`死亡`、`(可选)蓄力/胜利`。
- 切片:**1 把武器的全套**即可;完整版每把武器可有专属攻击/弹反手感。

## 三、Boss(每区 1 个,呼应轮回里的"人")
| Boss | 身份(剧情) | 战斗特点 | 优先级 |
|---|---|---|---|
| **试炼剑客 / 守门者** | 序章对手 | 快慢危混合连招谱、可练拼刀 | ⭐ |
| 师父 | 教你剑的人 | 招式像你、考验弹反基本功 | 后 |
| 挚友 / 同门 | 立场对立 | 双刀高速拼刀 | 后 |
| 叛徒 / 仇敌 | 害你之人 | 阴招/远程+近战混合 | 后 |
| **镜像之我 / 最终对手** | 反转核心(你自己) | 会用你的招、终局拼刀 | 后(结局) |
> Boss 动画量最大:多套攻击段 + 阶段切换 + 处决被杀演出 + 入场/血条。共 5–7 个。

## 四、NPC · 据点(Hub,讲故事 + 功能)
| NPC | 功能 | 动画 |
|---|---|---|
| **引导者 / 记忆守门人** ⭐ | 叙事推进、教学 | idle + 说话(2–3) |
| 铁匠 / 武器商 | 解锁/选武器 | idle + 说话 |
| 改造师 / 词条商 | 解锁弹反词条、元进度 | idle + 说话 |
| 神秘人 | 埋线、暗示真相/反转 | idle + 说话 |
| (可选)同伴/亡魂群像 | 氛围、支线碎片 | idle |
> NPC 轻量(站桩 + 说话),约 5–8 个。

## 五、其它实体 / 资产
- **弹道物**:箭 / 飞刀 / 法弹(可被弹反反弹)——小 sprite。
- **机关陷阱**:地刺、落石、火/毒区、移动平台——多在 tilemap 里。
- **拾取物**:魂/币、改造件图标、记忆碎片、血瓶/补给、钥匙。
- **环境/交互物**:篝火/存档点、神龛(风险献祭)、宝箱、门、传送/出口。
- **特效**(多数引擎做,部分可要素材):弹反火花、处决斩光、危攻击红光、冲击波、残影、死亡溶解、打击粒子。
- **UI**:血条、**架势条**(你 + 敌人)、武器/技能图标、改造件图标、Boss 血条、计时/死亡数、地图。
- **场景 tileset**:每生态区一套(规整瓦片,用 tilemap 编辑器);见各区主题。
- **音效/音乐**:挥砍、**弹反"叮"**、处决、受击、脚步、Boss 音乐、据点 BGM(`assets/sfx/` 钩子已留)。

## 六、规模概览(完整版目标)
- 主角 1(多武器皮)· 杂兵/精英 ~8–12 · Boss ~5–7 · NPC ~5–8
- 弹道/拾取/机关/UI 一批 · tileset 3–5 套(每生态区)· 一批音效音乐

## 七、垂直切片只需(先做这些)⭐
**主角(1 武器全套)+ 剑卒 + 重甲 +(可选弓手)+ 1 个 Boss + 1 个据点 NPC + 基础特效/UI/几个拾取**。
