# 苔光 · MOSSLIGHT — 设计文档

> 一只小林守，在黑暗合拢前一路冲刺、点亮孢心。
> Dash. Bounce. Relight the forest.

## 一句话
快节奏**冲刺平台跳跃**，主打"心流连段"与"瞬间重生、再来一次"，
生物荧光苔藓画风，适合速通与短视频传播。目标时长 **2–3 小时**。

## 世界观
**空苔森林 Hollowmoss** 靠「孢心」之光维持生机。**枯萎（the Blight）** 正在蔓延——
史莱姆是被腐化的孢生物，毒花扩散，光在一处处熄灭。
最后的小林守 **Viir**（蓝巫师）醒来，要在黑暗彻底合拢前，沿森林深入，逐一**点亮孢心**。
每个生态区 = 一颗大孢心；点亮即推进剧情、解锁新机制。

## 核心玩法
- **核心动词：冲刺 Dash**（素材自带 Dash 动画）。空中可冲刺一次。
- **刷新机制**：撞到**发光孢子/植物**或**踩到史莱姆**→ 冲刺重置 + 弹起。
  → 把冲刺一个接一个串起来，像飞一样穿过房间。这是"心流"与连段的来源。
- **危险**：毒花（Plant_8_Poison）、尖刺藤（Mossy 机关）→ 触碰即**瞬间重生**，零惩罚。
- **目标**：抵达**孢心（出口）**。可选收集发光孢，用于速通评级/隐藏路线。

## 结构（撑起 2–3 小时）
- ~35 个**短房间**（每个 20–60 秒），分 **4–5 个生态区**。
- 瞬间重生（无读条），房间内 checkpoint。
- **速通计时器** + 死亡数；通关后开放排行榜/计时模式。
- 难度曲线：教学(冲刺) → 弹史莱姆 → 连段 → 毒花/尖刺 → 综合考验。

## 为什么可能爆
1. 完美连段的丝滑 run 天生适合 Shorts/直播剪辑。
2. 生物荧光画风，随手截图就好看。
3. 易上手、难精通 + 计时排行榜 → 速通社区与"看我这把"文化。
