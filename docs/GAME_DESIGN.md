# 游戏设计 · 弹反拼刀 Roguelite(暂定名待定)

> 2D 像素横版 · Roguelite 动作 · **签名机制=弹反/拼刀(架势处决)** · 目标 Steam 体量。
> 主题(世界观/美术)**待定**,后期找素材时再定;本档先锁玩法。

---

## 1. 一句话
> 一款**以"弹反拼刀"为核心**的横版动作 Roguelite:不靠磨血,靠**精准弹反削架势→处决**;每局随机武器与"弹反流派"构筑,反复挑战 Boss 拼刀。

## 2. 核心战斗(灵魂)
- **架势条 Posture**:你和敌人都有。削满**敌人架势 → 处决**(秒杀型终结),而非慢慢砍血。
- **完美弹反 Parry**:敌人攻击命中前的窗口按格挡 → 弹开 + 大幅削敌架势 + **火花/顿帧/镜头推近**。
- **普通格挡**:挡住伤害但**涨自己架势**(自己架势满=被破防/硬直)→ 逼你去"弹"而不是"挡"。
- **拼刀**:敌人连段时连续完美弹反 = 火花四溅的过招,是招牌爽点与传播素材。
- **「危」攻击(红光预警)**:不可弹,必须**闪避/跳/特定应对**(下劈→格挡破、突刺→侧闪)→ 读招深度。
- **进攻**:轻/重攻击照常,但**真正击杀靠削架势处决**,鼓励主动过招。

### 手感关键(我们引擎已验证可做)
弹反判定窗口(几帧)+ 顿帧 hitstop + 火花 + 震屏 + 镜头微推 + 弹反音效。**这些决定爽不爽,垂直切片重点调这里。**

## 3. Roguelite 循环
> **选起始武器 → 闯随机关卡(打怪+捡构筑)→ 关末 Boss(拼刀大舞台)→ 进下一区(更难+新怪)→ 死亡 → 用资源永久解锁更多内容 → 再出发**
- **爽** = 弹反/处决手感;**上瘾** = 随机弹反流派滚雪球;**耐玩** = 每局不同 + 越解锁越丰富。

## 4. 两层构筑(build 全围绕"弹反")
1. **武器 = 定打法**:弹反手感/范围/处决各不同。
   - 例:太刀(窗口大、平衡)/ 双刀(拼刀快、窗口窄)/ 巨剑(弹反震退、慢)/ 伞(防御广、反弹强)/ 拳套(贴身、处决快)。
2. **改造词条 = 定流派**(都强化弹反,组 synergy):
   - 完美弹反回血 · 弹反放冲击波 · 弹反充能必杀 · 弹反后下一击必暴 · 连续弹反叠攻速 · **弹反把子弹反弹回敌人** · 处决回血/回闪 · 架势削满范围扩散 …
   - 流派雏形:**弹反流 / 处决流 / 反弹流 / 拼刀连段流**。
3. **风险层(可选)**:神龛"弹反窗口变窄但收益翻倍""少一半血但处决范围扩散"等,贪婪博弈。

## 5. 元进度(死亡后变强,撑时长)
攒"魂/币"永久解锁:**新武器、新弹反词条、新起始道具、角色被动、解锁新区域** → 丰富后续 run 的随机池。**内容靠组合爆炸,不靠手搓长流程。**

## 6. Steam 体量的内容规模(目标)
- 3–5 个**生态区**:每区一套敌人 + 主题机关 + **1 个 Boss**(拼刀谱)。
- ~15–25 把武器 · 50+ 弹反词条 · 10+ Boss/精英。
- **关卡 = 程序拼接**:用 `tools/tilemap` 做"房间模板",运行时随机拼成每局不同的关卡。

## 7. ⭐ 垂直切片(第一个要做的东西)
**先做一个能反复玩到爽的最小版,验证"拼刀爽不爽":**
- 1 角色:移动 / 跳 / 闪避(无敌帧)/ 轻攻击 / **格挡键(完美弹反窗口)** / 架势条。
- **2–3 种敌人**:① 有节奏连段(练弹反)② 带红光「危」(不可弹)③ 远程(弹反子弹)。
- **1 个 Boss**:一套连招让你练拼刀 + 处决。
- 3–4 个弹反词条,能组出明显流派差异。
- 一条短 run(出发→几间房→Boss→死亡→解锁一点→再来)。
- **达标标准:和这个 Boss 反复拼刀都觉得爽。** 爽了再扩内容;不爽就调手感。
- **可全程灰盒**(方块 + 火花 + 顿帧),不需要美术。
