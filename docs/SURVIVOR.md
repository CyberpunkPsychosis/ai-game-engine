# 幸存者模式（土豆兄弟式）· M1 核心循环（灰盒）

俯视角竞技场幸存者：**移动 + 自动瞄准开火 → 杀怪掉材料 → 捡材料(=钱+经验) →
升级三选一 → 波次倒计时结束 → 商店花材料 → 下一波(更强)**。

> 这是和现有「横版动作」并存的**新游戏模式**，复用引擎管线(`Juice/FX/AudioManager` 等)。
> M1 是**灰盒**：用色块跑通玩法，美术/动画/UI皮在 M3 用 `tools/sprite-forge` 批量补。

## 怎么运行（测试）

- Godot 编辑器里打开 `scenes/survivor/arena.tscn`，按 **F6**（运行当前场景）。
- 或临时把 `project.godot` 的 `run/main_scene` 指到该场景。
- 想从开始菜单进入，喊我加个按钮（避免动你现有菜单，先没接）。

## 操作

- **桌面**：`WASD` / 方向键移动。武器自动瞄准最近敌人开火。
- **手机**：屏幕任意处按住拖动 = 浮动摇杆移动（自动开火，整屏可拖）。
- 升级/商店弹窗会暂停游戏，点选即可。

## 文件地图（全在 `scripts/survivor/`）

| 文件 | 职责 |
|---|---|
| `arena.gd` | 总控：相机/边界/刷怪/波次计时/材料经验/升级/商店/结算（代码驱动搭场景） |
| `player.gd` | 8 向移动 + 属性表 `stats` + 多武器 + 受击/护甲/回血 |
| `weapon.gd` | 自动锁定最近敌人、按攻速冷却开火（伤害/攻速/射程吃 `stats`） |
| `projectile.gd` | 投射物，命中结算、穿透、寿命 |
| `enemy.gd` | 追玩家、贴身接触伤害、随波次成长、死亡掉材料 |
| `pickup.gd` | 材料宝石，进拾取范围吸向玩家 |
| `upgrades.gd` | 升级/商店选项数据（data 驱动） |
| `hud.gd` | 血条/经验/波次/计时/材料/击杀 + 升级三选一/商店/结算弹窗 |
| `vjoystick.gd` | 全屏浮动虚拟摇杆（手机） |

碰撞层沿用 `project.godot` 既有约定：layer2=player、layer3=enemy、layer4=player_hitbox、layer8=pickups。
