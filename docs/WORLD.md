# 刹那 TimeStop · 世界观 & 美术风格圣经（ART BIBLE）

> 这份文档有两个用途:① 定游戏的世界观/风格;② 当**训练 Scenario 风格模型的种子**
> （调色板 + 设计语言 + 提示词模板）。Pro 到位后照这套出**统一**美术。
> 原则:**设定为玩法服务**——每条 lore 都对应一个已实现的机制。

## 一、世界观

### Logline
在一个**被永远定格**的世界里，你是唯一还能动的人。靠"动"夺回时间，于绝境按下全场定格、翻盘。

### 设定:大凝滞 & 凝界
- 时间在此世界是**有形之物**。某一刹那，整个世界被冻结——史称 **「大凝滞」**。
- 此后的世界叫 **「凝界」**:雨滴悬在半空、落叶停在风里、火花凝固不灭。万物定格在那一瞬。
- 你是极少数 **「刹那者」**——能在凝界里自由行动、能用指尖**捏住"刹那"**(偷取/释放被冻的时间)。

### 敌人 = 死前一瞬的「残响」（对应三型敌人）
凝界里游荡的敌人是 **「残响」(Echoes)**:在大凝滞中被定格的存在，永远**重演死前最后一个动作**，
卡帧般抽搐、带轻微错位残影。三型各是一种"死亡瞬间":
| 机制型 | 名 | 是哪一瞬 |
|---|---|---|
| 冲锋 charger | **扑影** | 扑杀的那一瞬，永远向你冲 |
| 远程 shooter | **悬铳** | 扣下扳机那一瞬，释放悬停的弹 |
| 治疗 healer | **缚生** | 续命那一瞬，把同伴从死里拽回 |

### 机制 ↔ 设定 一一对应（这是这套设定的价值）
- **冻单体 / 全场定格** → 你是刹那者，本就能操弄凝固。全场定格 = 你短暂**重现大凝滞**。
- **能量靠进攻攒**（命中+8/击杀+28）→ 在静止的世界里，**唯有"动"才有力量**(称之为「势」)。越主动越有"势"去冻结。
- **闪避无敌帧** → 你"踏入刹那缝隙"，从定格的攻击间穿过。
- **被冻物可当踏板（未来涌现玩法）** → 你把敌人/子弹冻成实体，踩着它上天——"凝固之物即立足之地"。

### 进程钩子（以后扩展用）
凝界中心有 **「钟枢」**——大凝滞的源头。你一路向它推进，每层一个"被定格的大瞬间"(Boss=某个永恒之灾的化身)。最终抉择:**重启时间** 还是 **成为新的凝滞之主**。

## 二、美术风格圣经

### 总基调
**像素美术（16-bit+，厚重可读轮廓）** · 氛围参考 *Hyper Light Drifter* 的冷寂孤寂 +
*Dead Cells* 的像素动作可读性 + 一点**和风/禅意**(神社、钟楼、市井、悬停的雨)。

### ⭐ 核心视觉法则:冷暖二元（最重要、统一感的命根子）
- **凝界 = 冷**:蓝/青灰/霜白，死寂、低饱和。（已和现有后处理 shader 的"去色冷蓝"一致）
- **生命 / "势" / 你的动作 = 暖**:琥珀、赤红、暖金，作为**点缀**跳出来。
- **被冻物 = 纯冻蓝**（亮青）发光描边。
- 一句话:**世界是冷的，你是暖的，冻结是亮蓝的。** 任何素材都按这条配色。

### 调色板（锁色用 · 见 `docs/style/palette.png`）
冷（凝滞）: `#0b1622` `#15303f` `#2e5a73` `#5b97b5` `#9fd8e8` `#eaf6fb`
暖（生命/势）: `#7a1f2b` `#e63946` `#fb8500` `#ffb703` `#ffd166`
中性: `#22223b` `#4a4e69` `#9a8c98` `#c9c9d1`
高亮（冻结）: `#36e0ff`
> 生成像素素材时把这张 palette 喂给 Retro Diffusion 的 `inputPalette`，全套自动同色。

### 设计语言
- **主角（刹那者）**:剪影**极清晰**、单色黑描边、暖色主调（赤/琥珀）。轻甲+长衣摆（动起来有"动势"飘带）。
  脸不重要、轮廓和姿态重要。一手能凝出**亮蓝的"刹那"光痕**。
- **残响（敌人）**:冷色、半透/带**1–2px 错位残影**(卡帧感)，造型停在"死亡动作"的夸张极值。
  扑影=前倾扑姿；悬铳=举枪定格、枪口悬着弹;缚生=伸手拽人的姿态、缠绕的命线。
- **环境**:冷调凝固的东方市井/神社/钟楼。**点睛**:半空悬停的雨滴/落叶/火星(冷白小点)，
  暗示"时间停了"。视差分层、暗角、扫描线(已有后处理)。
- **VFX**:冻结=亮蓝霜纹包裹 + 轻脉动;全场定格=全屏去色冷蓝+暗角+白闪(已实现);命中=暖色火花。
- **UI**:深冷底 + 像素描边面板(已有)，能量条用**暖色**(势)，冻结/定格图标用**亮蓝**。中文用 `zpix`。

## 三、给 AI 出素材用（Pro 到位后照做）
1. **训风格模型**:用本圣经的调色板 + 5–15 张"定调图"(先用 Retro Diffusion 按下面提示词各出几张，挑最对味的)训一个 Scenario 风格模型 → 之后所有素材自动统一。
2. **提示词模板**(配 `inputPalette=palette.png`):
   > `"<主体>, side view, chunky 16-bit pixel art, cold stilled world, teal and frost blue palette with warm amber accents, crisp black outline, Hyper Light Drifter atmosphere, oriental, clean readable silhouette, transparent background"`
3. **优先清单**:主角(idle/run/jump/attack/dash 关键帧) → 三型残响 → 钟楼/神社 tileset(`RD Tile`) → 冻结/定格 VFX → UI 皮。
4. 动画走 `RD Animation` 或 `tools/sprite-forge`(整表生成→切帧)。

- **C. 暗黑奇幻**:哥特死寂教堂/血色月，更重口。冷灰 + 血红。
