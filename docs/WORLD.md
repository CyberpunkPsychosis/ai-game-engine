# 刹那 TimeStop · 世界观 & 美术风格圣经（ART BIBLE）

> 这份文档有两个用途:① 定游戏的世界观/风格;② 当**训练 Scenario 风格模型的种子**
> （调色板 + 设计语言 + 提示词模板）。Pro 到位后照这套出**统一**美术。
> 原则:**设定为玩法服务**——每条 lore 都对应一个已实现的机制。

## 一、世界观

### Logline
在一个**被永远定格**的世界里，你是唯一还能动的人。靠"动"夺回时间，于绝境按下全场定格、翻盘。

### 设定:大凝滞 & 凝界
- 时间在此世界是**有形之物**。某一刹那，整个世界被冻结——史称 **「大凝滞」**。
- 此后的世界叫 **「凝界」**:雨滴悬在半空、落叶停在风里、火花凝固不灭。万物定格在那一瞬。
- 你是极少数 **「刹那者」**——能在凝界里自由行动、能用指尖**捏住"刹那"**(偷取/释放被冻的时间)。

### 敌人 = 死前一瞬的「残响」（对应三型敌人）
凝界里游荡的敌人是 **「残响」(Echoes)**:在大凝滞中被定格的存在，永远**重演死前最后一个动作**，
卡帧般抽搐、带轻微错位残影。三型各是一种"死亡瞬间":
| 机制型 | 名 | 是哪一瞬 |
|---|---|---|
| 冲锋 charger | **扑影** | 扑杀的那一瞬，永远向你冲 |
| 远程 shooter | **悬铳** | 扣下扳机那一瞬，释放悬停的弹 |
| 治疗 healer | **缚生** | 续命那一瞬，把同伴从死里拽回 |

### 机制 ↔ 设定 一一对应（这是这套设定的价值）
- **冻单体 / 全场定格** → 你是刹那者，本就能操弄凝固。全场定格 = 你短暂**重现大凝滞**。
- **能量靠进攻攒**（命中+8/击杀+28）→ 在静止的世界里，**唯有"动"才有力量**(称之为「势」)。越主动越有"势"去冻结。
- **闪避无敌帧** → 你"踏入刹那缝隙"，从定格的攻击间穿过。
- **被冻物可当踏板（✅已实现 `player.gd:_stand_on_frozen`）** → 你把敌人/子弹冻成实体，踩着它上天——"凝固之物即立足之地"。全场定格时整个战场都成了立足点。

### ⭐ 双态结构:沉眠 ↔ 苏醒（2026-06 定稿,核心进程设计）
每个区域的残响默认处于 **「沉眠态」**——真的只会缓慢重演死前那个动作(循环、迟钝、几乎不构成战斗威胁)，
世界呈现真正的"凝固感"。此阶段玩法=**解谜/穿行**:冻结是解谜工具
(等残响循环到特定姿势的那一帧冻住=特定高度/位置的踏板;冻物挡弹=开路)。

区域深处有 **「醒钟」**——敲响它,本区域的时间被**局部重启**,残响全部**苏醒**
(进入完整战斗 AI),玩法切换为**爽快打斗**。

**敲不敲,玩家自己选**:
- **敲醒**:硬仗 + 完整奖励(掉落/能量经济全开)。
- **不敲**:残响保持沉眠,可潜行解谜通过,但**奖励大减**;且能量靠命中/击杀攒——
  不打=能量饥荒=全场定格用不起 → 自然形成"高技巧低资源"的潜行流派,不是懒人模式。
  (可补不敲独占:隐藏 lore / 仅沉眠态可用的踏板路线 / 速通成就。)

### 进程钩子 & 主题闭环
凝界中心有 **「钟枢」**——大凝滞的源头。你一路向它推进，每层一个"被定格的大瞬间"(Boss=某个永恒之灾的化身)。
最终抉择:**重启时间** 还是 **成为新的凝滞之主**。
**分形呼应**:每个房间的"敲醒钟与否"就是终局抉择的微缩预演——玩家一路都在练习这个选择,
走到钟枢时,这个抉择已有肌肉记忆与情感重量。

## 二、美术风格圣经

### 总基调
**像素美术（16-bit+，厚重可读轮廓）** · 氛围参考 *Hyper Light Drifter* 的冷寂孤寂 +
*Dead Cells* 的像素动作可读性 + 一点**和风/禅意**(神社、钟楼、市井、悬停的雨)。

### 颜色:暂不锁定(2026-06 用户定)
> 旧"冷暖二元铁律 + 锁色调色板"方案已删(git 历史可找回)。
> **现阶段不规定颜色**:素材按工具默认/素材自带配色来,先把游戏做出来;
> 等素材量起来、看到整体观感后再决定要不要统一色调。
> (后处理 shader 的定格去色冷蓝是**机制表现**不是美术规定,保留。)

### 设计语言
- **主角（刹那者）**:剪影**极清晰**、描边干净。轻甲+长衣摆（动起来有"动势"飘带）。
  脸不重要、轮廓和姿态重要。一手能凝出"刹那"光痕。(配色未定)
- **残响（敌人）**:半透/带**1–2px 错位残影**(卡帧感)，造型停在"死亡动作"的夸张极值。(配色未定)
  扑影=前倾扑姿；悬铳=举枪定格、枪口悬着弹;缚生=伸手拽人的姿态、缠绕的命线。
- **环境**:凝固的东方市井/神社/钟楼。**点睛**:半空悬停的雨滴/落叶/火星，
  暗示"时间停了"。视差分层、暗角、扫描线(已有后处理)。
- **VFX**:冻结=霜纹包裹+轻脉动;全场定格=全屏去色+暗角+白闪(已实现, 机制表现非美术规定);命中=火花。
- **UI**:像素描边面板(已有)。中文用 `zpix`。(配色未定)

## 三、给 AI 出素材用（Pro 到位后照做）
1. **训风格模型**:用本圣经的调色板 + 5–15 张"定调图"(先用 Retro Diffusion 按下面提示词各出几张，挑最对味的)训一个 Scenario 风格模型 → 之后所有素材自动统一。
2. **提示词模板**(色板参数暂不喂,颜色未定):
   > `"<主体>, side view, chunky 16-bit pixel art, crisp black outline, clean readable silhouette, clean readable silhouette, transparent background"`
3. **优先清单**:主角(idle/run/jump/attack/dash 关键帧) → 三型残响 → 钟楼/神社 tileset(`RD Tile`) → 冻结/定格 VFX → UI 皮。
4. 动画:Scenario 出视频动作 → `tools/sprite-keyer` 选帧/抠图/对齐/导横条(见 `SPRITE_PIPELINE.md`);瓦片走 `RD Tile`。
