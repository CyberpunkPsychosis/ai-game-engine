# 🎨 画画学习线(写给"不会画画"的你)

> 好消息:**做 2D 游戏美术,真不需要传统画功。** 两条路组合用——
> **像素画**(靠逻辑摆像素,零基础友好)+ **AI 出图**(你项目的核心,解决一致性瓶颈)。

---

## 路线 A:像素画(零基础首选)
像素画的核心不是"画得像",是"理解像素和邻居怎么互动",更像解谜/搭积木,适合理工脑。

### 工具
- **Aseprite**(行业标准,付费但便宜,Steam 有售) —— 强烈推荐,省时功能多。
- 免费替代:**LibreSprite**(Aseprite 开源分叉)、**GraphicsGale**、**Piskel**(网页版)。

### 教程(按顺序)
1. [Derek Yu · 像素画基础](https://www.derekyu.com/makegames/pixelart.html)
   ——《洞穴探险》作者写的经典入门。
2. [Pedro Medeiros · 绝对新手入门 #1](https://medium.com/pixel-grimoire/how-to-start-making-pixel-art-2d1e31a5ceab)
   —— 手把手,从 16×16 开始。
3. [Lospec 教程合集](https://lospec.com/pixel-art-tutorials)
   + 它的[免费调色板库](https://lospec.com/palettes) —— **新手直接用现成配色,别自己调色**。
4. [Kiwi 的 7 步学像素画](https://2dwillneverdie.com/blog/kiwis-7-step-program-for-learning-pixel-art/)

### 关键心法
- 从 **16×16** 小图起步,一张 5–30 分钟能完成,别一上来画大图。
- **先用现成调色板**,把精力放在形状和明暗,不要纠结配色。
- 一两个像素错位就能毁掉一张图——像素画靠**耐心和方法**,不是天赋。

### 练习目标
- [ ] 画一个 16×16 的小角色站立图
- [ ] 给它画 3 帧走路动画(站/抬左脚/抬右脚)
- [ ] 画一组地形瓦片(草地/泥土),能在引擎里铺成地面

## 路线 B:AI 出图(你项目的核心方向)
你文档里写过,真正的瓶颈是 **"AI 出美术素材的一致性"**。这条线就是攻这个。

### 教程 / 工具
1. [HuggingFace · 2D 资源生成(游戏 AI 系列)](https://huggingface.co/blog/ml-for-games-4)
   —— 系统讲怎么用 AI 出游戏资源。
2. [游戏开发者的 AI 美术工具对比 2025](https://aloa.co/ai/comparisons/ai-image-comparison/top-ai-art-tools-game-developers)
3. [Spritesheets.ai · AI 游戏资源生成指南](https://www.spritesheets.ai/blog/ai-game-asset-generation-guide)
   —— 专门做"多帧动画角色一致"。

### 解决"一致性"的三板斧(最关键)
1. **LoRA**:拿少量统一风格的图,训练一个小模型,让 AI 锁定你的专属画风。
   —— 这是专业管线对抗"AI 出图风格飘"的主力武器。
2. **image2image 迭代**:别指望一次文生图完美。AI 出草图 → 你/AI 手动修 → 再喂回去迭代。
3. **开局先定 style guide**:把**配色 / 线条风格 / 细节程度 / 视角**写死成一份文档,
   之后所有素材都对齐它。风格统一,一半靠规范,不全靠模型。

### 现实预期
- AI 出图能解决大约 **70%**,"每一帧完全一致"目前仍需 LoRA + 人工修补,**没有银弹**。
- 这也是为什么"会一点像素画"很重要——能自己补那 30%,整条链路就活了。

### 练习目标
- [ ] 跑通一次完整工作流:文生图 → 图生图迭代 → 导出可用素材
- [ ] 用 LoRA 或风格锁定,让同一个角色的多张图保持风格一致
- [ ] 写一份你自己游戏的 style guide(一页纸即可)

---

## 给自己的节奏建议
- 画画**每天一点点**(15–30 分钟)就行,别集中猛攻——它靠手感积累。
- 像素画和 AI 出图**不冲突,要合用**:AI 出大方向,你用像素画修细节/补一致性。
- 早期别追求好看,**先追求"能用"**——丑但能动的游戏 > 漂亮但永远做不完的素材。
