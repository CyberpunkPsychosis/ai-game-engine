上传给 Claude

选文件 → 上传，会提交到仓库 incoming/，Claude 就能读到。
zip 会在浏览器里自动解压后逐个文件上传，所以大压缩包（如几十 MB 的素材包）也能传。
没令牌？点这创建（见下方步骤）。令牌只存你手机本地，不会上传。
怎么创建令牌（1 分钟）
  1. 打开上面「点这创建」
  2. Token name 随便填；Expiration 选 30 天
  3. Repository access 选 Only select repositories → 勾 ai-game-engine
  4. Permissions → Repository permissions → Contents 设为 Read and write
  5. 底部 Generate token → 复制那串 → 粘到上面输入框
上传成功后，回聊天告诉 Claude「传好了」。
